Efron's Dice

I developed this code using Java, this code proves the mathematical reasoning behind Efron's Dice. Efron's dice is an example of intransitive dice. In the example to the left, A > B > C > DIn terms of rolling higher more than 50% of the time. This would lead us to believe that A > D, however this is not true. TL;DR: there is always a die in the set that can roll higher than the die we roll.
